# Code of Conduct

IronCache adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
by reference as its code of conduct. Everyone participating in the project,
in any space the project uses, is expected to follow it.

To report unacceptable behavior, contact the maintainers privately at
conduct@ironcache.dev. Reports are handled confidentially. The full text of the
Contributor Covenant 2.1, including the enforcement guidelines, is available at
the link above.
